# 数据中心 - 数据为0诊断报告

## 问题现象

用户反馈以下数据显示为0：
1. ✅ **智能辅助时长** = 0
2. ✅ **安全评分** = 0  
3. ✅ **前车急刹** = 0
4. ✅ **碰撞预警** = 0

## 根因分析

### 1. 智能辅助时长为0 ⚠️

#### 数据来源
```python
# calc_drive_stats.py
engaged = False  # 初始值

# 从以下字段读取激活状态：
if w == "carState":
    cs_en = getattr(cs, "cruiseState", None)
    if cs_en is not None and hasattr(cs_en, "enabled"):
        engaged = cs_en.enabled

elif w == "carControl":
    if hasattr(cc, "enabled"):
        engaged = cc.enabled

elif w == "selfdriveState":
    if hasattr(ss, "enabled"):
        engaged = ss.enabled

# 积分计算辅助时长
if engaged:
    self.assisted_m += ds_m
    self.assisted_duration_s += dt_s
```

#### 可能原因

| 原因 | 概率 | 诊断方法 |
|------|------|----------|
| **qlog中engaged字段一直为False** | 高 | 查看C3实际是否启用了openpilot |
| **qlog格式版本不兼容** | 中 | 检查 LogReader 是否能解析 |
| **segment太短，未激活** | 低 | 查看segment时长 |
| **sunnypilot分支字段名不同** | 高 | 可能用了其他字段名 |

#### 修复建议

**方案A：增加调试日志（推荐）**
```python
# calc_drive_stats.py 增加统计
engaged_count = 0
total_count = 0

for msg in lr:
    total_count += 1
    if engaged:
        engaged_count += 1

print(f"DEBUG: engaged率 = {engaged_count}/{total_count} = {engaged_count/total_count*100:.1f}%", file=sys.stderr)
```

**方案B：兼容更多字段**
```python
# 兼容不同分支的字段名
engaged = False
if w == "carState":
    cs = msg.carState
    # 尝试多个可能的字段
    if hasattr(cs, "enabled"):
        engaged = cs.enabled
    elif hasattr(cs, "cruiseState"):
        cruise = cs.cruiseState
        if hasattr(cruise, "enabled"):
            engaged = cruise.enabled
        elif hasattr(cruise, "active"):  # 兼容旧版本
            engaged = cruise.active
```

### 2. 安全评分为0 ❌ BUG

#### 根因
```python
# calc_drive_stats.py line 478 - 强制覆盖为0！
daily["safetyScore"] = compute_safety_score(daily)

# 但 compute_safety_score 返回值被计算后又设为0
def to_daily_drive_stats(self):
    return {
        ...
        "safetyScore": 0,  # ← 这里强制设为0
    }
```

**这是脚本BUG**，应该让脚本计算后直接使用：

```python
# 修复：移除初始化时的0
def to_daily_drive_stats(self):
    return {
        ...
        # "safetyScore": 0,  # 删除这行
    }

# 在输出时统一计算
daily["safetyScore"] = compute_safety_score(daily)
```

#### 验证方法
1. 手动在C3上运行脚本查看stderr输出
2. 检查 `/data/appdata/drive_stats.json` 中 `safetyScore` 值

### 3. 前车急刹/碰撞预警为0 ⚠️

#### 数据来源

**方式A：从 onroadEvents 读取**
```python
elif w == "onroadEvents":
    for ev in msg.onroadEvents:
        cur.add(str(ev.name))
    for name in EVENT_MAP:  # fcw, aeb等
        acc.register_edge(name, name in cur)
```

**方式B：从 radarState/modelV2 推断**
```python
# 雷达前车
elif w == "radarState":
    tailgate, stationary, slow, has_any = _process_radar_leads(rs, v_ego)
    
# 视觉前车（BYD无雷达）
elif w == "modelV2":
    if not acc.has_radar_leads:  # 仅当雷达不可用时使用视觉
        tailgate, stationary, slow = _process_modelv2_leads(mv, v_ego)
```

#### 可能原因

| 原因 | 概率 | 说明 |
|------|------|------|
| **onroadEvents中没有fcw/aeb事件** | 高 | 正常驾驶可能真的没触发 |
| **事件名称不匹配** | 中 | sunnypilot可能用了不同的事件名 |
| **qlog中没有这些字段** | 低 | 旧版本可能缺失 |
| **前车检测未生效** | 中 | radarState和modelV2都未检测到前车 |

#### 验证方法

**手动检查qlog**：
```python

```

#### 修复建议

**增加事件兼容性**：

## 诊断步骤

### Step 1: 确认有数据

### Step 2: 手动运行脚本查看详细输出
```bash

```

查看 stderr 输出：
- `[全量扫描完成] 处理 X 个 segment` - 是否真的处理了数据
- 是否有 `跳过 XXX: 错误` - 解析失败

### Step 3: 检查输出JSON
```bash
cat /data/appdata/drive_stats.json | head -50
```

查看字段值：
```json
{
  "date": "2026-07-15",
  "totalDistanceKm": 45.3,      // 应该 > 0
  "assistedDistanceKm": 0.0,    // ← 如果是0，说明engaged一直为false
  "assistedDurationMinutes": 0, // ← 同上
  "safetyScore": 0,             // ← 这个是BUG，应该被计算
  ...
}
```

### Step 4: 采样单个segment手动解析
```python
cd /data/openpilot
python3 -c "
from tools.lib.logreader import LogReader
import sys

qlog = '/data/media/0/realdata/<最新segment>/qlog.zst'
lr = LogReader(qlog)

engaged_count = 0
total_count = 0
event_count = {}

for i, msg in enumerate(lr):
    if i >= 5000:  # 采样前5000条
        break
    
    w = msg.which()
    total_count += 1
    
    if w == 'carState':
        cs = msg.carState
        if hasattr(cs, 'cruiseState'):
            cruise = cs.cruiseState
            if hasattr(cruise, 'enabled') and cruise.enabled:
                engaged_count += 1
    
    if w == 'onroadEvents':
        for ev in msg.onroadEvents:
            name = str(ev.name)
            event_count[name] = event_count.get(name, 0) + 1

print(f'engaged率: {engaged_count}/{total_count} = {engaged_count/total_count*100:.2f}%')
print(f'事件统计: {event_count}')
"
```

## 附录：字段依赖关系

```
qlog消息类型 → calc_drive_stats.py → DriveStats → UI显示

carState.cruiseState.enabled 
  → acc.integrate(engaged=True)
  → assistedDurationMinutes
  → 智能辅助时长 (圆形进度)

onroadEvents[].name in {"fcw", "stockFcw"}
  → acc.register_edge("fcw")
  → collisionWarning
  → 碰撞预警 (卡片)

onroadEvents[].name in {"aeb", "stockAeb"}
  → acc.register_edge("aeb")
  → leadCarEmergencyBrake
  → 前车急刹 (卡片)

takeovers + warnings
  → compute_safety_score()
  → safetyScore
  → 安全评分 (圆形进度)
```
