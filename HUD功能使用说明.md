# C3摄像头+HUD叠加功能使用说明

## 🎉 功能介绍

现在你的SunnyPilot Toolbox App可以：
- ✅ 实时查看C3摄像头画面（MJPEG 480p @ 8fps）
- ✅ 叠加显示HUD信息（车速、档位、转向角、前车距离等）
- ✅ 完美兼容Android 7.0车机
- ✅ C3性能开销低（CPU增加<8%）
- ✅ 延迟100-200ms，流畅查看

---

## 📱 使用步骤

### 1. 连接到C3设备
1. 打开SunnyPilot Toolbox App
2. 进入"设备连接"界面
3. 输入C3的IP地址和SSH密码
4. 点击"连接"

### 2. 进入实时视频界面
1. 连接成功后，进入"实时摄像头"功能
2. 选择摄像头：
   - **主视角**：正前方ACC/车道保持视角
   - **广角**：两侧变道/盲区视角
3. 等待2-3秒启动

### 3. 查看HUD信息
启动后你会看到：
- **左上角**：车速（km/h）+ 档位
- **右上角**：转向角度
- **中上方**：前车距离（带颜色提示：绿=安全，黄=注意，红=危险）
- **中央**：横向控制状态（绿色✓表示启用）
- **两侧**：转向灯指示（绿色圆圈）
- **底部**：车道线位置（黄色线条）
- **顶部中央**：警告信息（如有）
- **右下角**：刹车指示（红色圆圈）

### 4. 切换HUD显示
- 点击**左上角的"HUD"按钮**可以开关HUD叠加层
- 👁️ 图标：HUD显示中
- 👁️‍🗨️ 图标：HUD已隐藏

## 🎮 HUD信息说明

### 显示内容

| 位置 | 信息 | 说明 |
|------|------|------|
| 左上角 | 车速 | 实时速度（km/h），大字体白色显示 |
| 左上角下方 | 档位 | P/R/N/D/S等，青色显示 |
| 右上角 | 转向角 | 方向盘转向角度（°） |
| 中上方 | 前车距离 | 前车距离（米），颜色根据距离变化 |
| 中央 | 控制状态 | "✓ 横向控制"表示openpilot已接管 |
| 左右两侧 | 转向灯 | 绿色圆圈+箭头指示 |
| 底部 | 车道线 | 黄色竖线显示车道线位置 |
| 顶部 | 警告 | 系统警告信息（如有） |
| 右下角 | 刹车 | 红色圆圈显示刹车状态 |

### 颜色含义

**前车距离颜色：**
- 🟢 绿色：>30米，安全距离
- 🟡 黄色：15-30米，注意保持距离
- 🔴 红色：<15米，距离过近！

**警告信息颜色：**
- ⚪ 白色：普通提示
- 🟡 黄色：需要用户注意
- 🔴 红色：严重警告，需要立即处理
